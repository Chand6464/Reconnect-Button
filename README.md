# Reconnect Button (1.21.11)

A lightweight Fabric mod for Minecraft 1.21.11 that adds a "Reconnect" button to the Disconnected screen. No more going back to the multiplayer menu just to rejoin your favorite server!

## ✨ Features
- **One-Click Rejoin:** Adds a button to the disconnect screen to immediately reconnect to the last server you were on.
- **Smart Positioning:** Perfectly aligned with the vanilla "Back to Server List" button.
- **Lightweight:** Designed to be compatible with performance mods like Sodium, Lithium, and Iris.

## 🛠 Installation
1. Ensure you have the **Fabric Loader** installed for Minecraft 1.21.11.
2. Download the latest version of the **Fabric API** for 1.21.11.
3. Drop the `reconnect-button.jar` and the Fabric API into your `.minecraft/mods` folder.
4. Launch the game!

## 🏗 Building from Source
If you want to build the mod yourself:
1. Clone the repository.
2. Open a terminal in the folder.
3. Run `./gradlew build`
4. Find your jar in `build/libs/`.
